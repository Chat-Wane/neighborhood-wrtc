    public disconnect(id: string | undefined): * source

    Disconnect one of the arc with the identifier in argument. If it was the last arc with such id, the socket is relocated to the dying table. The socket will be destroy after a bit. If there is no argument, disconnect the whole.

    Params:

    NameTypeAttributeDescription
    id string | undefined

    Id provided to just disconnect the arc or if undefined disconnect all arcs
