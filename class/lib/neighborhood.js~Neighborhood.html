    public register(protocol: IProtocol): INeighborhood source

    The protocolId asks this module to get an interface. The interface comprises functions such as connect, disconnect, or send. ProtocolId should provide some functions as well such as failed, opened, received, pid.

    Params:

    NameTypeAttributeDescription
    protocol IProtocol

    The protocol that requires the interface.

    Return:

    INeighborhood

    The interface to use this module easily.

    private _connect(protocolId: string, arg1: function | object, arg2: object) source

    Params:

    NameTypeAttributeDescription
    protocolId string

    The identifier of the protocol that wishes to establish an new connection.
    arg1 function | object

    Either a callback function to send the message to the remote peer (for instance, it can use a signaling server or the already created WebRTC connexions), or a message received from the remote peer.
    arg2 object

    The message received from a peer that initialized a WebRTC connection.
